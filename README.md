# Ahmad's Cookbook

A living cookbook of software development practices, architecture patterns, and data engineering solutions, published at <https://ahmadhatahet.github.io/cookbook>.

The site is generated by a single dependency-light Python script — no framework. The landing page is an auto-generated, clickable mind-map SVG of all categories and articles; every article page gets a heading-based table of contents, a site-wide index, a "you are here" locator map, and buttons to download as PDF, copy the raw markdown link, or copy a share link with citation.

## Adding content

1. Drop a markdown file into a category folder (e.g. `automation/my-article.md`):

   ```markdown
   ---
   title: "My Article Title"
   ---

   Content with headings, code blocks, images, tables...
   ```

2. Push to `main`. The GitHub Action rebuilds and deploys the site — the mind map, indexes, and navigation update automatically.

New categories are just new top-level folders containing `.md` files. To give a folder a nicer display name, add it to `CATEGORY_NAMES` in `build.py`; otherwise the folder name is title-cased automatically.

## Local development

```bash
./serve.sh          # build + preview at http://localhost:8000 (./serve.sh 3000 for another port)
uv run build.py     # build only, into _site/
```

## Layout

- `build.py` — the whole generator (markdown rendering, mind-map SVG, indexes); site settings are in the config block at the top
- `templates/` — the stylesheet and toolbar/scroll-spy JavaScript, copied to `_site/assets/`
- `about.md` — the About page
- `citation.txt` — template for the "Share & cite" button; placeholders: `{author}` `{year}` `{title}` `{site_title}` `{url}` `{category}`
- `.github/workflows/publish.yml` — build and deploy to GitHub Pages
